# Stories – CRM SBCE

Esta pasta contém as **stories** (tarefas) que a Squad AIOS executa. A ordem de execução é definida na **instrução única** que você cola no Cursor (ou Claude Code).

## Como usar

1. **Crie ou edite** os arquivos de story aqui (ex.: `STORY-001-contas-a-pagar.md`).
2. **Abra** `INSTRUCOES-COMPLETAS-SQUAD-AUTONOMA.md` e copie o **Texto para colar**.
3. **No Cursor:** novo chat (Agent ou Chat) → cole o texto → envie.
4. A squad executa as stories na **ordem listada** na instrução e devolve um resumo ao final.

## Lista de stories na instrução

A lista “STORIES (ordem obrigatória)” dentro de `INSTRUCOES-COMPLETAS-SQUAD-AUTONOMA.md` define quais stories serão feitas. Atualize essa lista quando adicionar ou priorizar novas stories.
